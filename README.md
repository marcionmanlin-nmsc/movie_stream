# Movie Streaming Site

A responsive movie streaming site inspired by yts.bz. Uses a color theme based on the provided logo (blue, yellow, red).

## Features
- Responsive grid of movies with thumbnails, titles, and short descriptions
- Movie detail page with poster, full description, and embedded Facebook trailer
- Mobile-friendly and visually modern

## How to Use
Open index.html in your browser. Replace placeholder data and images with real movie data as needed.

---

**Note:**
- Replace placeholder images and Facebook trailer links in `script.js` and `movies.js` with real data as needed.
- All code is in the `moviesite` folder.
